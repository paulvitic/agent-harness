# Skills

This directory stores skill-related agent harness content.

## Intended structure

- Create one folder per skill.
- Place the skill instructions in `SKILL.md`.
- Keep bundled resources next to the skill, such as:
  - `scripts/`
  - `templates/`
  - `references/`

## Use skills for

- Repeatable workflows that benefit from bundled assets
- Task-specific instructions that should load on demand
- Reusable operational or domain-specific helper material
