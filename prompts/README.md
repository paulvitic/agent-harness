# Prompts

This directory stores prompt templates and prompt assets.

## Intended structure

- Store reusable prompt templates in markdown or text files.
- Group related prompt packs with any supporting notes or examples.
- Keep prompts concise, task-oriented, and easy to compose with agents, skills, or instructions.

## Use prompts for

- Starter prompts for common tasks
- Prompt libraries for repeatable workflows
- Supporting context that helps standardize agent interactions
